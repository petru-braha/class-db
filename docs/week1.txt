- ddl commands - for the database schema
- dml commands - for views and tabels
  - SELECT
  - INSERT
  - DELETE
  - UPDATE

- `;`
  - does not store the statement in the buffer
  - automatically runs the statement

- '\'
  - stores the statement in the buffer
  - does not flush the buffer
  - transactions
  - PL/SQL blocks
